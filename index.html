Mistico (concept)

Style sheet customisation language.





Getting started

setbg + color
Sets background color.

setfg + color
Sets foreground color.

setg1 + color
Sets first gradient color.

setg2 + color
Sets second gradient color.

setf1 + font-family
Sets titles font.

setf2 + font-family
Sets webpage font.

setn + text
Sets title.

setb + text
Sets subtitle.

light
Sets solarised light mode.

default
Brings back to default settings.
